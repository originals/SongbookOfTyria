## Features

| Feature | Description |
|---------|-------------|
| **Song Library** | Browse an extensive collection of music tabs |
| **Tab Notation** | View detailed notation for each song |
| **Search & Filter** | Find songs by name, arranger, or category |
| **Sorting Options** | Sort tabs by various criteria |
| **Piano Keybinds** | Customize your instrument keybinds |
| **Audio Playback** | Listen to MP3 previews of songs |
| **Favorites** | Mark your favorite tabs for quick access |
| **Guild Authentication** | Access exclusive guild content |

---

## Installation

1. Install [Blish HUD](https://blishhud.com)
2. Download `SongbookOfTyria.bhm` from the releases page
3. Place the file in `Documents/Guild Wars 2/addons/blishhud/modules`
4. Launch Blish HUD and enable Songbook of Tyria in the module settings

---

## Song Library

Access the song library via the Songbook corner icon. Browse through available tabs, search for specific songs, or filter by category.

| Action | Description |
|--------|-------------|
| Search | Type in the search box to filter songs |
| Filter | Use category filters to narrow results |
| Sort | Click column headers to sort |
| Favorite | Click the heart icon to mark favorites |

---

## Tab Viewer

Click on any song to open the detailed tab view with full notation display.

| Feature | Description |
|---------|-------------|
| Notation Display | musical notation |
| Auto Scroll | Navigate through longer pieces |
| Fontsize | Adjust notation size for readability |
| Color Coding | Notes are color-coded for easier reading |

## Audio Playback

Listen to MP3 previews when available to learn how the song should sound.

| Control | Description |
|---------|-------------|
| Play/Pause | Toggle playback |
| Progress | Seek through the track |
| Volume | Adjust playback volume |
